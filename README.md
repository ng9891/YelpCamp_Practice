# YelpCamp_Practice

[Link to the course](https://www.udemy.com/the-web-developer-bootcamp/)

Dependencies:
- ExpressJS
- MongoDB (mongoose)
- EJS
- Bootstrap 4
- Body-parser
- Method-Overwrite
- Express-session
- Passport
- Connect-Flash
