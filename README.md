# YelpCamp_Practice
Learning/Practicing ExpressJS following an Udemy course.  
[Link to the course](https://www.udemy.com/the-web-developer-bootcamp/)
***
Dependencies:
- ExpressJS
- MongoDB (mongoose)
- EJS
- Bootstrap 4
- Body-parser
- Method-Overwrite
- Express-session
- Passport
- Connect-Flash
- MomentJS
- Leaflet Map
